## 示例一：题材定位卡

```
【题材定位卡】
- 书名（暂定）：《我的系统能无限回档》
- 题材/标签：都市异能、系统流、悬疑、轻喜剧
- 核心梗：社畜程序员穿越到平行世界，获得"死亡回档"系统——每次意外死亡都会回到三天前。利用这个能力破解连环悬案、识破商业阴谋，最后发现自己的死亡并非偶然。
- 目标读者画像：20-35岁男性上班族，喜欢悬疑+系统流爽文
- 竞品参考书：《死亡回归》《异常生物见闻录》
- 卖点/差异化：主角不是战斗型，是靠智商和情报差破局；悬疑元素贯穿始终，不是纯爽文
- 预计总字数：150-200万字
- 更新频率：日更4000字
```

## 示例二：世界观设定（都市异能类）

```
【世界观设定】
- 时代背景：2025年，平行世界华夏国，科技水平与现实一致
- 核心设定：存在"异常者"——极小概率觉醒特殊能力的人，占总人口百万分之一
  - 异常者被国家秘密机构"第11局"管理
  - 大多数异常者能力微弱（强化视力、短暂预知等）
  - 主角的系统是独一无二的"回档"级能力
- 社会结构：普通人不知道异常者的存在；异常者社会中有一套灰色交易体系
- 地理：主舞台为"江海市"——虚构的东部沿海超大城市
- 势力分布：
  - 官方：第11局（管理异常者，维护秩序）
  - 灰色：地下异常者交易市场
  - 反派：某利用异常者进行犯罪活动的跨国财团
```

## 示例三：角色卡

```
【角色卡】
- 姓名：林越
- 身份/标签：28岁程序员 / "死亡回档"能力者
- 外貌特征：178cm，偏瘦，戴黑框眼镜，长相普通但眼神锐利
- 性格：表面社恐宅男，实则观察力极强，危机时刻异常冷静
  - MBTI: INTJ
  - 核心特质：善于复盘和分析，讨厌不确定性
- 背景故事：原世界是普通程序员，加班猝死后穿越到平行世界同名者身上
- 能力/金手指：【无限回档】——每次死亡后回到72小时前的锚点
  - 被动触发，死亡后自动激活
  - 保留全部记忆
  - 锚点会在每次回档后更新
  - 限制：不能主动存档，死亡间隔期不能超过72小时
- 动机/目标：第一阶段（活下去并适应能力）→ 第二阶段（查清自己穿越/死亡真相）→ 第三阶段（瓦解幕后黑手）
- 成长弧线：逃避→利用→掌控→超越
- 关键关系：搭档苏晚晴（第11局调查员）——从互不信任到生死与共
- 说话风格：日常是带点吐槽的随意，关键时刻简短有力
  - 口癖："这剧情我熟"（当预知到危险时）
```

## 示例四：章节正文（节选）

> 以下为第1章开场示例，风格为轻悬疑+都市：

---

林越盯着电脑屏幕上那行报错代码看了整整三分钟。

他又死了一次。

这是第几次了？十七？十八？他已经懒得数了。每次死亡都被弹回这个时间点——周一上午十点十七分，办公桌上那杯咖啡还冒着热气，显示器右下角弹出第三十七封未读邮件。

"林越，方案下午两点前给我。"组长老周的声音从背后传来。

"行。"他头也不回地答。

不是他态度差，是他真的不知道该用什么表情面对一个他已经"见过"三十多次的人。上一轮这位老周死在他面前——被某种透明的、看不见的东西拧断了脖子。那个画面像一帧截图一样卡在他脑子里。

林越深吸一口气，端起咖啡喝了一口。

凉的。每次都凉的。

他放下杯子，手指在键盘上敲了几下，关掉报错窗口，打开了一个空白文档。如果他的记忆没错——他的记忆从来没错——再过四十分钟，前台会送来一个包裹。包裹里是一部旧手机。手机里有一条短信，只有四个字：

"又见面了。"

然后他的死亡倒计时就开始了。

林越推了推眼镜，开始在文档里打字：

*第18次回档观察记录*
*已知死亡原因：3种*
*已知凶手：0人*
*已知真相：0%*

*本次策略变化：不再被动等线索来找我。我要主动去找那个寄手机的人。*

他敲下回车键，保存文件，关掉文档。动作流畅自然，一看就是已经做过十七次操作。

"林越。"旁边工位的同事探头过来，"中午一起吃饭？食堂新开了个窗口。"

他转过头，露出一个恰到好处的社畜微笑。

"不了，我有约。"

——他约了那个送快递的。

---

### 本章要点分析
- **开场钩子**：主角"又死了一次"——立即制造悬念
- **世界观暗示**：通过细节自然透露出"循环/回档"设定，不靠旁白说明
- **角色塑造**：用动作（敲文档、推眼镜）和微妙的内心OS塑造角色
- **章尾钩子**："约了那个送快递的"——一句话制造期待
- **去AI味体现**：短句多、口语化、无过多连接词、"社畜微笑"有梗感
