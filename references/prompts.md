# 网文写作 Prompt 模板库

## 一、开书阶段

### 1.1 灵感拓展

```
我是一个想写网文的新手，我有一个灵感种子：{一句话灵感}。

请帮我做三件事：
1. 评估这个灵感的价值（亮点、风险、差异化空间）
2. 给3个不同的展开方向（每个方向一句话概括+核心卖点）
3. 推荐最适合的题材和风格
```

### 1.2 世界观构建追问

```
我正在构建一个{题材}世界观，核心设定是{核心设定}。

请从以下维度依次提问，每次只问一个维度，等我想好再问下一个：
1. 力量体系/科技树
2. 社会结构
3. 历史背景
4. 地理分布
5. 核心冲突根源

每个问题要保持开放，给2-3个选项供选择，而不是直接给我答案。
```

### 1.3 人物设定生成

```
请根据以下世界观和故事梗概，生成一套网文角色组：

【世界观简述】：{世界观}
【故事梗概】：{梗概}
【题材】：{题材}

请输出：
1. 主角（1人）：姓名、核心特质、金手指/能力、成长弧线、说话风格
2. 重要配角（3人）：姓名、与主角关系、叙事功能、性格特点
3. 反派（1-2人）：姓名、动机、能力、与主角的冲突点

每个角色控制在100字以内。要求：人设新颖，避免套路化模板。
```

## 二、大纲与规划

### 2.1 全书大纲生成

```
请为一部{题材}网文生成全书大纲。

【书名】：{书名}
【核心梗】：{核心梗}
【预计字数】：{字数}
【主要角色】：{角色列表}
【世界观核心】：{世界观核心}

输出格式：
- 使用三幕式或打怪升级式结构
- 每卷标明：卷名、核心事件、字数范围、主要爽点、结尾钩子
- 总卷数控制在3-7卷
- 标注每卷的节奏定位（铺垫期/爆发期/转折期/收尾期）
```

### 2.2 章节细纲生成

```
【当前卷信息】
- 卷名：{卷名}
- 本卷目标：{目标}
- 已写章节数：{N}
- 剩余章节数：{M}

【写作状态】
- 当前进度：{剧情走到哪里}
- 需要解决的伏笔：{伏笔列表}
- 即将引入的新要素：{新角色/新能力/新势力}

请生成接下来5章的章纲，每章包含：
1. 开场钩子（一句话）
2. 核心情节（3-4个要点）
3. 爽点/高潮
4. 章尾钩子
5. 字数建议
```

## 三、正文生成

### 3.1 标准章节生成

```
【写作指令】
- 任务：生成第{章号}章正文
- 题材：{题材}
- 风格：{轻松/热血/悬疑/甜宠...}
- 视角：{第一/三人称}
- 本章字数：{2000-3000}
- 节奏提示：{铺垫/高潮/转折/收束}

【本章章纲】
{章纲内容}

【核心设定速查】
{世界观+角色+力量的当前状态}

【前文风格参考】（可提供一段前文作为风格锚点）
{前文段落}

请直接输出正文，不要附加说明。注意保持与前文风格一致。
```

### 3.2 战斗场面

```
【战斗场景生成】
- 参战角色：{角色A} vs {角色B}
- 战斗背景：{场景/原因}
- 力量体系：{核心规则}
- 战斗目标：{胜负条件/叙事目的}
- 篇幅：{800-1500字}

写作要求：
1. 战斗要有节奏变化（试探→交锋→危机→爆发→结局）
2. 不要只用"速度很快"来描述，用具体画面
3. 技能/招式的释放要有代价或限制
4. 战斗结果要推动剧情或角色成长
5. 避免回合制对波式的单调描写
```

### 3.3 情感/日常场景

```
【日常/情感场景生成】
- 场景类型：{日常互动/感情升温/冲突和解}
- 参与角色：{角色列表}
- 场景背景：{前情提要}
- 篇幅：{1000-2000字}

写作要求：
1. 用对话和细节推进，减少心理描写独白
2. 符合角色的性格和说话风格
3. 情感要有层次感，不要一上来就浓烈
4. 穿插轻松幽默调剂
5. 场景结尾要有情感余韵或伏笔
```

## 四、风格控制

### 4.1 一键提取文风特征

```
请分析以下文本的文风特征，提取关键要素供后续生成参考：

{样本文本}

提取维度：
1. 语言风格（书面/口语/文白夹杂）
2. 句式特点（长短句比例、排比/反问等修辞偏好）
3. 用词习惯（高频词、特色词汇、避免的词汇）
4. 叙事节奏（快速推进/细腻描写/夹叙夹议）
5. 对话风格（简洁/啰嗦/犀利/温柔）
6. 描写偏好（重环境/重动作/重心理）
7. 情绪基调（热血/温馨/压抑/诙谐）

输出为结构化格式，供后续注入提示词使用。
```

### 4.2 风格配置文件格式

当用户定义了一种写作风格时，以以下结构保存：

```yaml
# 风格配置文件
name: "示例：轻快打脸风"
language_style: "口语化为主，略带网文腔"
sentence_pattern: "短句为主，关键战斗用排比增强气势"
vocab_preferences: ["特么", "玩意儿", "整", "妥妥的"]
vocab_avoid: ["然而", "因此", "综上所述", "换言之"]
narrative_pace: "快节奏，三句话内进入剧情"
dialogue_style: "简洁犀利，带方言口癖"
description_focus: "重动作和对话，略环境描写"
emotional_tone: "热血诙谐，压不住的正能量"
avoid_patterns: ["不要大段心理独白", "不要上帝视角评价"]
```

## 五、优化与校验

### 5.1 去AI味终极检查清单

```
请对以下文本进行AI味检测并修改。

检测维度（每项打分1-5，5=最像AI）：
1. 逻辑连接词过多（然而/但是/因此/毕竟）___
2. 总分总结构明显 ___
3. 形容词堆砌 ___
4. 对话过于完整合乎语法 ___
5. 段落间过渡刻意 ___
6. 缺少口语化的语气词和省略 ___
7. 情绪描写过于"标准" ___

对得分3以上的项目，逐项修改。
修改后输出最终版本，附带修改说明。
```

### 5.2 一致性检查

```
请检查以下新生成的章节是否与已有设定一致：

【已有设定】
- 世界观：{世界观}
- 角色状态：{各角色当前状态}

【新章节内容】
{新章节正文}

检查项：
1. 角色性格 vs 言行：有无OOC（角色崩坏）
2. 角色能力/境界：是否与设定一致
3. 时间线：有无前后矛盾
4. 伏笔照应：是否有需要收但没收的线
5. 地点/空间：是否存在逻辑矛盾

对发现的问题列出清单，标注严重程度（高/中/低）和修改建议。
```

### 5.3 爽点密度检查

```
请分析以下章节的爽点密度：

分析维度：
1. 本章爽点数量：___
2. 爽点类型分布（打脸/升级/收获/反转/情感）：___
3. 距离上一个爽点的字数间隔：___
4. 本章最高潮位置（百分比）：___
5. 章尾钩子强度（1-10）：___

网文经验法则：
- 每2000字至少1个小爽点
- 爽点间隔不超过1500字
- 高潮在60-80%位置最佳
- 章尾钩子强度不低于7/10

如果未达标，给出修改方案。
```
